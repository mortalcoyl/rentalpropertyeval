# Rental Property Outcome Calculator

A Vercel-ready Next.js app for comparing a rental property purchase against a market investment.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy to Vercel

Push this folder to GitHub, then import the repo in Vercel. Vercel should auto-detect Next.js.

Or deploy from the command line:

```bash
npm install -g vercel
vercel --prod
```
